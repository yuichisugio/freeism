## 適用範囲

本文書は、freeism プロジェクトのリポジトリ、そのコミュニティ活動（Issues、Pull Requests、Discussions、関連するオンラインまたはオフラインの場）に適用されます。プロジェクトやコミュニティを代表して行動する場合（公式の連絡手段の利用、公式アカウントでの発信、イベントでの参加など）にも適用されます。

## 私たちの約束

オープンで歓迎的な環境を育むために、貢献者およびメンテナーは、年齢、体型、障害の有無、民族性、性自認や表現、経験の程度、国籍、外見、宗教、性的指向にかかわらず、すべての人に対してハラスメントのない体験を提供することを約束します。

## 私たちの基準

前向きな環境づくりに寄与する行動の例は次のとおりです。

- 誰もが参加しやすい、歓迎的でインクルーシブな言葉遣い
- 異なる視点や経験への敬意
- 建設的な批評を寛容に受け入れる姿勢
- コミュニティ全体の利益を最優先に考える姿勢
- 他のコミュニティメンバーへの思いやり

不適切な行動の例は次のとおりです。

- 性的な言語表現や、不快な性的関心や誘い
- 荒らし、侮辱または中傷的なコメント、個人攻撃または政治攻撃
- 公的または私的なハラスメント
- 明示的な許可なく他者の個人情報（物理的または電子的な住所など）を公開すること
- 職業的な場において合理的に不適切と見なされるその他の行為

## メンテナーの責務

メンテナーは、許容できる行動の基準を明確にし、不適切な行動があった場合には適切かつ公平な是正措置を講じます。メンテナーは、この行動規範に沿わないコメント、コミット、コード、Wiki の編集、Issue、その他の貢献を削除・編集・拒否する権利と責任があります。また、不適切・脅迫的・攻撃的・有害と判断した行動に対して、貢献者を一時的または恒久的にアクセス禁止とする場合があります。

本行動規範を誠実に遵守・運用しないメンテナーは、プロジェクトの他のリーダーシップメンバーの判断により、一時的または恒久的な措置の対象となる場合があります。

## 通報と対応

ハラスメントや不適切な行為を目撃した、または経験した場合は、次のいずれかの方法で報告してください。

- 本リポジトリの Issues に「code-of-conduct」ラベルを付けて報告する（個人情報は記載しないでください）
- GitHub の非公開通報フォームから報告する: [Report abuse](https://github.com/contact/report-abuse)
- メンテナー（リポジトリオーナー）に直接連絡する（GitHub プロフィールに記載の連絡手段をご利用ください）

すべての報告に対して、状況に応じて必要かつ適切な対応が行われ、報告者のプライバシーは可能な限り尊重されます。運用上の詳細方針がある場合は、別途お知らせします。
